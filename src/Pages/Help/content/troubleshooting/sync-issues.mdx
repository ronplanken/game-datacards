## Sync Issues

### Sync is not working

If sync is not updating across devices:

1. **Check your connection** — The sync indicator in the header shows your current status. "Offline" means no server connection.
2. **Check your subscription** — Cloud sync requires a Premium or Creator subscription. If your subscription expired, sync is paused until you renew.
3. **Check sync is enabled** — Each category and datasource must have sync individually enabled. Look for the sync icon next to each item.
4. **Try refreshing** — A page refresh re-establishes the real-time connection to the server.

### Sync shows "Error"

An error status usually means the server rejected an operation. Common causes:

- **Subscription limit reached** — You have more synced items than your tier allows. Remove some items from sync or upgrade your plan.
- **Network timeout** — A temporary connection issue. The system will retry automatically.

### Sync shows "Conflict"

See the Resolving Conflicts article for detailed instructions on handling conflicts.

### Changes not appearing on other devices

Real-time sync has a small delay (about 2 seconds) before changes are uploaded. If changes are not appearing:

1. Make sure both devices are online and logged in.
2. Check that both devices are using the same account.
3. Verify the items in question have sync enabled on both devices.
4. Refresh the page on the receiving device.
