## Printing

### Cards look different when printed

The print view uses a separate layout optimized for paper. Some differences from the on-screen preview are normal:

- **Colors may vary** — Screen colors (RGB) and print colors (CMYK) can differ. This is a limitation of the color model conversion.
- **Margins** — Your browser's print dialog adds margins by default. Set margins to "None" or "Minimum" for the best results.
- **Background graphics** — Most browsers do not print background colors and images by default. Enable "Background graphics" in your print dialog settings.

### Print dialog settings

For the best print results:

| Setting | Recommended value |
|---------|-------------------|
| Margins | None |
| Scale | 100% (do not fit to page) |
| Background graphics | Enabled |
| Headers and footers | Disabled |

### Cards are cut off or too small

If cards are being clipped or appear too small:

1. Set the print scale to **100%** — do not use "Fit to page" as this can shrink the cards.
2. Set margins to **None** to maximize the printable area.
3. Check your paper size matches what you expect (A4, Letter, etc.).
