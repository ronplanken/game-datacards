## Enabling and Disabling Sync

### Enabling sync

To start syncing an item, click the <Icon name="RefreshCw" /> sync icon next to it. The item is uploaded to your cloud account immediately.

If the item was previously synced to a different account, you will see a confirmation dialog asking whether you want to claim it for your current account.

### Disabling sync

To stop syncing, click the <Icon name="RefreshCw" /> sync icon again. You will be asked whether to:

- **Keep cloud copy** — The item stays in the cloud but stops receiving updates from this device.
- <Icon name="Trash2" /> **Delete cloud copy** — The item is removed from the cloud. Your local copy is not affected.

### What happens to local data

Disabling sync never deletes your local data. The item remains in your browser exactly as it was. Only the cloud connection is removed.

### Re-enabling sync

You can re-enable sync at any time. If the cloud copy still exists, the system checks for conflicts between your local version and the cloud version before syncing.
