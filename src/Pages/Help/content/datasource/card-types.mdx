## Card Types

Card types define the kinds of cards in your datasource. Each one is built on a base type that determines its structure:

- <IconLabel icon="Swords" label="Unit" /> — Stats, weapons, and abilities. Use this for datasheets, warscrolls, or fighter cards.
- <IconLabel icon="BookOpen" label="Rule" /> — A title, description, and a list of nested rules.
- <IconLabel icon="Sparkles" label="Enhancement" /> — Custom fields plus a keywords collection.
- <IconLabel icon="Zap" label="Stratagem" /> — A title, phase, and description.

Add, remove, and reorder card types using the controls in the left panel.
