## Metadata

The metadata editor controls supplementary information that appears on unit cards — keywords, faction keywords, points, and banner display. These settings apply to all cards of a given unit card type.

### Metadata properties

| Property | Description |
|----------|-------------|
| <IconLabel icon="Tag" label="Keywords" /> | Enable a keywords section on the card (e.g., "Infantry", "Character", "Psyker"). Each card gets a keywords input where you can add multiple keyword tags. |
| <IconLabel icon="Tags" label="Faction Keywords" /> | Enable a separate faction keywords section (e.g., "Adeptus Astartes", "Ultramarines"). Displayed separately from regular keywords. |
| **Points** | Enable a points cost field on each card. |
| **Points format** | How points are displayed: **per model** (shows cost per individual model) or **per unit** (shows total unit cost). |

### Age of Sigmar banner properties

AoS base systems add banner configuration:

| Property | Description |
|----------|-------------|
| **Banner type** | Controls the banner at the top of the card: **faction** (uses faction color), **custom** (uses custom text), or **hidden** (no banner). |
| **Banner text** | Custom text displayed in the banner when banner type is set to "custom". |

### Datasource-level metadata

The datasource itself also has metadata that you can edit by selecting the datasource name in the left panel:

| Property | Description |
|----------|-------------|
| **Name** | Datasource name (e.g., "My Custom Army"). Max 200 characters. |
| **Version** | Version string (e.g., "1.0.0"). Max 50 characters. |
| **Author** | Creator name. |
| <IconLabel icon="Palette" label="Header color" /> | Color used for card headers and accent elements. |
| <IconLabel icon="Palette" label="Banner color" /> | Color used for card banners and backgrounds. |

These colors are applied to all cards in the datasource and give your custom cards a consistent visual identity.
