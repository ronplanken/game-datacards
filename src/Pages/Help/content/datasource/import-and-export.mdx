## Import and Export

<Icon name="Download" /> **Export** your datasource as a JSON file to create a backup or share it with others. The export includes your field definitions, metadata, and faction colors — but not individual card data.

<Icon name="Upload" /> **Import** a JSON file to load a datasource structure. Drag and drop the file onto the import dialog or use the file picker. The file is validated automatically — you'll see any problems listed before anything changes.
