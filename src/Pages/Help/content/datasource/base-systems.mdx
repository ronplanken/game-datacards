## Base Systems

The base system you choose when creating a datasource sets the starting point for your card types:

- **Warhammer 40K 11th Edition** — Pre-configures stat lines, weapon types, and ability categories in the 40K datacard format, and seeds the full 11th Edition keyword glossary (weapon keywords and core abilities) so cards show rule explanations out of the box.
- **Warhammer 40K 10th Edition** (legacy) — The same 40K card structure with the 10th Edition weapon keyword set.
- **Age of Sigmar** — Sets up warscroll-style fields and structure.
- **Blank** — No presets. You define every field from scratch.

The base system also determines which visual style is used when previewing your cards.
