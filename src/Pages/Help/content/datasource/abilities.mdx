## Abilities

Set up ability categories for your unit cards. Categories group abilities by type — you might have Core Abilities, Faction Abilities, and Wargear, or any grouping your game system needs.

### Category properties

Each ability category has these settings:

| Property | Description |
|----------|-------------|
| **Key** | Internal identifier (e.g., `core`, `faction`). Must be unique. |
| **Label** | Display name used in the card data editor. |
| **Header** | Optional header text displayed above this category on the card. Leave empty to hide the header. |
| **Format** | How abilities are displayed. See formats below. |
| **Layout** | Width of the category block: **full** (100%), **half** (50%), **third** (33%), or **quarter** (25%). Use smaller widths to place categories side-by-side. |

### Display formats

| Format | Description |
|--------|-------------|
| **Name only** | Shows just the ability name (e.g., "Deadly Demise D3"). Use this for simple keyword-style abilities. |
| **Name and description** | Shows the ability name with a rules description below it. Use this for abilities that need explanation text. |

### Age of Sigmar category properties

AoS base systems add two extra properties per category:

| Property | Description |
|----------|-------------|
| **Phase** | Enable a per-ability phase text field (e.g., "Combat Phase"). |
| **Color** | Enable a per-ability banner color for visual grouping. |

### Default 40K ability categories

The 40K preset creates four categories:

- **Core** — name-only format (e.g., "Deadly Demise D3", "Feel No Pain 5+")
- **Faction** — name and description format
- **Unit Abilities** — name and description format
- **Damaged** — name and description format, with a "Damaged" header
