## Sections

Sections let you add content blocks to unit cards for things like Wargear Options, Unit Composition, or Loadout. Each section appears as a titled block on the card. Sections are optional — you can have zero or as many as you need.

### Section properties

Each section has three settings:

| Property | Description |
|----------|-------------|
| **Key** | Internal identifier (e.g., `wargear-options`). Must be unique. |
| **Label** | Display name shown as the section title on the card (e.g., "Wargear Options"). |
| **Format** | How the section content is rendered. See formats below. |

### Display formats

| Format | Description |
|--------|-------------|
| **List** | Renders the content as a list of items, similar to how wargear options appear on official datacards. Each item is a separate line. |
| **Rich text** | Renders the content as free-form text with formatting. Suitable for longer descriptions or mixed content. |

### Adding and managing sections

Add, remove, and reorder sections using the controls in the section editor. The order in the editor determines the order on the card.

### Default 40K sections

The 40K preset includes three sections:

- **Wargear Options** — list format
- **Unit Composition** — list format
- **Loadout** — rich text format
